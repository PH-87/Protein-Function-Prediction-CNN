## Overview
This project was conducted during the UNLV Summer Research Program.
The objective was to investigate how different protein sequence encoding methods affect enzyme function prediction using CNN and Transformer models.

## My Contributions

- Proposed comparison of multiple encoding methods
- Implemented CNN model in PyTorch
- Designed preprocessing pipeline
- Conducted experiments and performance evaluation
- Wrote the final report
- Presented research findings in English
